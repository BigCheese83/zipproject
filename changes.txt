v.2.1 - 04/08/2018

- GUI на Swing
- Вывод консольных сообщений в компонент JTextArea

v.2.0 - 02/05/2018

- Рефакторинг с применением Java NIO2 и Java 8 Stream API

v.1.0 - 13/03/2015

- Первая версия
- Запуск из консольной строки
    java -jar zipproject-X.X.jar [полный путь для архивирования].
- для сжатия zip используется Deflater.BEST_COMPRESSION
- подсчет количества сжатых файлов/папок и вывод на консоль
- Конфигурационный файл для исключений .zipignore. Поддерживаются следующие шаблоны:
    # - комментарий
    {hidden} - исключаются все скрытые файлы
    {file} - исключаются все файлы
    {dir} - исключаются все папки
    [папка]/ - исключается папка и все находящиеся в ней файлы/папки
    маска * - любой символ или его отсутствие
