ZipProject - консольная утилита для архивирования исходников на java в формате zip.
Данная утилита помогает быстро формировать архив с исходниками без вспомогательных файлов, наподобие генерируемых IDE, системами автоматической сборки, и пр.
Основная особенность - автоматическое управление процессом архивирования с помощью конфигурационного файла .zipignore, наподобие того, как это реализовано в системах контроля версиями Git, SVN.
В этом файле можно задать шаблон файлов или директорий, которые не будут включаться в архив.

Рекомендуемые требования - JRE v.1.7

Предварительно, в той же папке с программой, создать файл .zipignore.
Если этого не сделать, будут архивированы все файлы.
Указать шаблоны файлов/папок, который будут игнорированы при архивировании.
Поддерживаются следующие шаблоны:
# - комментарий
{hidden} - исключаются все скрытые файлы
{file} - исключаются все файлы
{dir} - исключаются все папки
[папка]/ - исключается папка и все находящиеся в ней файлы/папки
маска * - любой символ или его отсутствие

Пример использования:
java -jar zipproject-X.X.jar [полный путь для архивирования]
